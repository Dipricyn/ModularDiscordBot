Class: NoticeMePlugin

NoticeMePlugin()

This plugin supports playing a sound randomly within a timespan in the channel of a randomly chosen user. Depends on the soundboard plugin to run! It reads the property "noticeMeTimer": from each sound object in the soundboard plugin's config file.

Constructor

new NoticeMePlugin()

Example
{
    sounds: [
        {
            "command": "mrgl",
            "files": ["sounds/murloc0.mp3"],
            "noticeMeTimer": 3600000
        }
    ]
}
